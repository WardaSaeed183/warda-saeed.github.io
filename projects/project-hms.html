HMS (Hotel Management System)

A comprehensive SQL database designed to be the backend engine for modern hotel operations.

Project Type: Database Architecture
Tech Stack: Microsoft SQL Server, SQL
GitHub: View Schema/Code

The Goal

Hotels require a robust and reliable system to manage a constant flow of data related to guests, rooms, and services. The objective of this project was to design and implement a normalized SQL database from the ground up, creating a scalable backend capable of handling all core hotel operations, from booking and check-in to final billing.

Project Context

This was a semester-long database project developed in a team of two. My partner, Muaarij, and I collaborated closely on all aspects of the design and implementation, ensuring the final product was both functionally complete and well-architected for our final grades.

Our Collaborative Contributions

We shared the workload equally, tackling key challenges together:

  • Database Design & ERD: We jointly designed the Entity-Relationship Diagram (ERD), mapping out all tables and their relationships to accurately model hotel operations.
  • Schema Implementation: We both worked on writing the SQL DDL scripts to create the tables, constraints, and indexes in Microsoft SQL Server, ensuring data integrity through proper normalization (3NF).
  • Backend Logic (Stored Procedures): We divided the work of writing the core stored procedures. I focused on the procedures for billing and room service charges, while my partner handled room booking and availability checks.
  • Testing & Data Population: We collaboratively created test data and ran queries to validate that our database functioned correctly under various scenarios.

Challenges & Lessons Learned

Working in a pair on a complex database design was a fantastic learning experience in communication and shared ownership. The main technical challenge was ensuring our schema was perfectly normalized to prevent data anomalies while still being efficient for complex queries. This project provided a deep, practical understanding of how a well-designed database serves as the critical foundation for any successful application.
