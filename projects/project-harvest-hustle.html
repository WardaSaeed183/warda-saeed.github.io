Harvest Hustle

A C++ console game demonstrating the practical application of core Data Structures & Algorithms.

Platform: Console Application
Tech Stack: C++, Data Structures
GitHub: View Source Code

The Challenge

The primary objective of this project was to apply and implement fundamental data structures from scratch in a practical context. The challenge was to architect a console-based game using Trees and Linked Lists to manage core logic like level progression, scoring, and dynamic item generation.

Project Context

This was a semester project developed with excellent teamwork by a group of three. We collaborated closely on all phases, from initial design to final implementation, ensuring a successful and functional result.

My Role & Contributions

  • Linked List Implementation: I took ownership of implementing the Linked List data structure from scratch in C++. This structure was used to efficiently manage the randomly generated fruits and obstacles appearing on screen.
  • Game Loop & Scoring Logic: I designed and coded the main game loop, which included the logic for player scoring, handling user input, and triggering "game over" conditions when negative-point items were collected.
  • Collaborative Algorithm Design: I worked with my teammates to design the Tree structure used for managing the game's difficulty levels and overall progression.

Challenges & Lessons Learned

The most insightful challenge was implementing the Tree and Linked List data structures manually in C++ without relying on standard libraries. This forced a deep understanding of pointers, memory management, and node traversal. Collaborating effectively as a team to debug complex algorithmic issues was a fantastic experience in communication and shared problem-solving. This project was invaluable for bridging the gap between theoretical computer science concepts and their practical application.
