EaseToCure Website

A user-centric web platform simplifying complex medical information into a clear and accessible format.

Platform: Web Application
Tech Stack: Node.js, Firebase, HTML/CSS, JavaScript
GitHub: View Source Code

The Problem

Finding understandable and relevant information about medicines online is a significant challenge. Patients and caregivers are often confronted with dense medical jargon, making it difficult to quickly grasp essential details. This information overload can lead to confusion and anxiety, highlighting the need for a simpler, more user-friendly solution.

Project Context

This was a semester-long group project built by a team of four members. As the team lead, I was responsible for task delegation, backend architecture, and ensuring the final integration resulted in a cohesive and high-quality product that met our academic requirements.

My Role & Contributions

  • Team Leadership & Backend Architecture: I led the project, assigned tasks to each team member, and designed the complete backend using Firebase for authentication and Firestore for real-time data management.
  • Core Backend Implementation: I personally developed the Firebase functions and security rules to handle all data related to medicines and user accounts.
  • Integration & Optimization: After individual components were completed by team members, I merged all parts of the project, optimized the codebase by removing redundancies, and linked the entire front-end to the backend services I had built.
  • Critical UI/UX Overhaul & Feature Implementation: Recognizing that key screens needed refinement to meet project standards, I took initiative to rebuild and enhance several critical components overnight. This included:
    • Redesigning the Medicine Details, Search by Category, and Search by Disease screens.
    • Implementing universal search functionality across the application.
    • Adding the "Forgot Password" feature and search suggestions on the homepage.

Leading the team while also being a core developer was a significant challenge that taught me invaluable lessons in project management and communication.
